## Pages
<details>

- 下载[_worker.js](https://github.com/eaqx/edgetunnel/blob/main/_worker.js)文件 移入文件夹
- 进入Cloudflare的"Workers and Pages" 新建Pages项目 上传文件夹并部署
- 点击"设置" 进入"变量和机密" 设置PROXYIP和UUID
- 再次上传文件并部署使变量生效
  
</details>
